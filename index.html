2. Click Extensions → Apps Script. Delete all code. Expand "View Apps Script Code" above, copy it, and paste it in.

3. Click Deploy → New Deployment. Type: Web App. Execute as: Me. Access: Anyone. Click Deploy and copy the URL.

4. Paste the URL in the field above and click Connect.
